Relative Paths

How to link to an asset in child folder:

background-image: url('images/image.jpg');

How to link to an asset in parent folder:

background-image: url('../image.jpg');

Combine the two to traverse filepaths:

background-image: url('../images/image.jpg');

Flexbox

Flexbox: Overview

ï HTML/CSS page layout using floats was always a hack.

ï Flexbox, a new layout mode in CSS3, is a tool for making 2D layouts (e.g. column vs. row) easier with a ton more options

ï Main idea: give a container the ability to alter (flex) its childrens' sizing to best-fill the available space

ï Bonus: containers can easily vertically/horizontally align their children

Flexbox: Overview

ï Two components: a flex container and its flex items (children)

Flexbox: Overview

ï The equivalent in HTML:


      <div class="container">
        <div class="item"></div>
        <div class="item"></div>
        <div class="item"></div>
      </div>

ï The <div class="container"> represents the flex container

ï The <div class="item">'s represent the flex items (children)

Flexbox: Overview

ï To use flexbox, apply display: flex; as a CSS rule to the flex container

ï Once applied, all of the flex container's immediate children become flex children

ï The flex children should now be next to eachother (although we can change this with the flex-direction property

ï Codepen example

Flexbox - how it works

ï Set the css property display: flex on the *parent/container* of the elements you want to position

ï Set additional properties on the container, or individual elements of the elements you want to position to modify positioning/size as you wish. (some properties are exlusive to the container and some on the flex items inside the flex container.)

Flexbox Containers: On-axis Alignment

ï Use the justify-content CSS property to horiztonally align a flex container's children:

⇒ flex-start (default): items align at the left

⇒ flex-end: items align at the right

⇒ center: items are centered

⇒ space-between: items are evenly distributed with items aligning at the conatiner's start/end

⇒ space-around: items are evenly distributed in the line with equal space around them

Flexbox Containers: Off-axis Alignment

ï Use the align-items CSS property to vertically align a flex container's children:

⇒ flex-start: items aligned at the top of the container

⇒ flex-end: items aligned at the bottom of the container

⇒ center: items are centered

⇒ baseline: items are aligned along the baseline of their respective text

⇒ stretch (default): items stretch to fill their container
